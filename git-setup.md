# Git Setup Instructions

After installing Git, run these commands in your terminal:

## 1. Initialize Git Repository
```bash
git init
```

## 2. Add Remote Repository
```bash
git remote add origin https://github.com/exburn35/ekimdemirci.git
```

## 3. Add All Files
```bash
git add .
```

## 4. Create Initial Commit
```bash
git commit -m "Initial commit: Ekim Demirci SEO Website"
```

## 5. Set Main Branch (if needed)
```bash
git branch -M main
```

## 6. Push to GitHub
```bash
git push -u origin main
```

## Alternative: If you need to authenticate
If GitHub asks for authentication, you can either:
- Use a Personal Access Token (recommended)
- Or use GitHub Desktop application
